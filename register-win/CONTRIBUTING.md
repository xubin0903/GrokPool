## 🔧 代码贡献

### 开发环境准备

1. **Fork 本仓库**
2. **克隆到本地**：
   ```bash
   git clone https://github.com/你的用户名/grok-register-win.git
   ```
3. **安装依赖**：
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```

### 提交 PR

1. **创建分支**：`git checkout -b feature/your-feature-name`
2. **做出修改**并测试
3. **提交**：`git commit -m 'feat: 添加 xxx 功能'`
4. **推送**：`git push origin feature/your-feature-name`
5. **提交 Pull Request**

### 提交规范

遵循 [Conventional Commits](https://www.conventionalcommits.org/zh-hans/v1.0.0/)：

- `feat:` 新功能
- `fix:` Bug 修复
- `docs:` 文档变更
- `refactor:` 代码重构
- `chore:` 构建/工具变更

### 代码风格

- Python 代码遵循 PEP 8
- 函数和类添加必要的中文注释
- 新增依赖需更新 `requirements.txt`
